STORE:
Es el centro y la verdad de todo, con métodos para actualizar, obtener y escuchar datos.
  1. Contiene el estado de la aplicación
  2. Puedes acceder al estado con el metodo getState()
  3. Puedes actualizar el estado con el método dispatch()
  4. Escucha cambios en el método subscribe(listener)
  5. Dejas de escuchar cambios retornando la función del método subscribe(listener)
